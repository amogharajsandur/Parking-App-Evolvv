# Smart Parking App - Backend

A modular Node.js/Express backend for the Smart Parking application.

## Structure
- `/config`: Database configuration
- `/controllers`: Business logic
- `/routes`: API endpoints
- `server.js`: Entry point

## Setup
1. `npm install`
2. Create `.env` file with `DATABASE_URL` (PostgreSQL)
3. `npm run dev`

## Deployment
Deploy to **Render** or **Heroku**. Ensure `DATABASE_URL` is set in environment secrets.
